# Abbeydale YardOps Command Centre

Mobile-first installable PWA prototype for Abbeydale Construction yard operations.

## What this demo shows

- Branded mobile dashboard
- Plant and asset register
- Site allocations
- Maintenance planner
- Open site requests
- Supplier and invoice lookup
- Demo AI-style operational search
- Add-to-home-screen app behaviour

## How to publish with GitHub Pages

1. Open repository settings.
2. Go to **Pages**.
3. Under **Build and deployment**, select **Deploy from a branch**.
4. Select `main` and `/root`.
5. Save.
6. Open the GitHub Pages URL on mobile.
7. In Chrome, tap the menu and choose **Install app** or **Add to Home screen**.
